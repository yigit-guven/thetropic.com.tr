# Security Policy

## Supported Versions
Only the latest version of this codebase is supported with security updates.

## Reporting a Vulnerability
**DO NOT** create a public GitHub issue for security vulnerabilities.

Please email security reports to: contact@yigitguven.net

## Response Time
We will acknowledge receipt within 48 hours and provide a more detailed response within 5 business days.

## Scope
This security policy applies only to the code in this repository. The actual business operations and customer data are handled through secure external channels (WhatsApp, email, phone).
