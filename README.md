# The Tropic - Premium Tropical Fruits Website

A modern, responsive e-commerce website for The Tropic - Premium Tropical Fruits, built with pure HTML, CSS, and JavaScript.

This website's values can be modified from Google Sheets. Here is an example [Google Sheets link](https://docs.google.com/spreadsheets/d/1pa4FL2LKmaqkhayt2iLMTQXf8HQoznAg7WDg0AXRfAM/edit?usp=sharing).

## Features

- **Multi-language Support** - Turkish, English, German, and Russian
- **Responsive Design** - Works perfectly on all devices
- **Shopping Cart** - Browser-based cart functionality
- **Contact Forms** - Integrated WhatsApp and email communication
- **No Backend Required** - Static site hosted on GitHub Pages
- **Legal Compliance** - Comprehensive Privacy Policy and Terms of Service

## Getting Started

### Prerequisites

- A modern web browser
- GitHub account (for deployment)

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Open index.html in your browser or use a local server

### Deployment

This website is designed to be deployed on GitHub Pages:

1. Fork this repository
2. Go to repository Settings → Pages
3. Select source branch (usually main or master)
4. Your site will be available at https://yourusername.github.io/Tropiko-Website

## Customization

### Adding New Products
1. Add product HTML in index.html
2. Update product information in JavaScript fruitInfo object
3. Add translations for all supported languages
4. Add product image to images/ folder

### Adding New Languages
1. Add language option in HTML
2. Add translations in JavaScript translations object
3. Add flag SVG data
